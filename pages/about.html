Idea

Saya (Juan) memiliki hobi kuliner dan memasak dan ingin memutuskan untuk berbagi cerita kepada pembaca tentang kuliner di seluruh dunia. Saya yakin masih sangat banyak kuliner di seluruh dunia, namun saya berharap pembaca dapat betah melihat resep - Juan Patrick

Design

Dengan pengalaman kurang lebih 6 bulan, saya bekerja sebagai Front End sekaligus berkolaborasi bersama dengan UI/UX menciptakan desain produk yang baik. Saya suka merancang dan mendesain sebuah tampilan dengan pengalaman dan referensi untuk membuat web Resepi.

Building

Web Resepi dibangun menggunakan HTML, CSS, dan JavaScript. Juan membangun web dengan bantuan materi tambahan dari kelas Dicoding - Belajar Dasar Pemrograman Web

Testing

Selama proses pembangunan, web Resepi diuji dari Front End Testing dengan memastikan web secara fungsional dan responsif layar bekerja dengan baik

Deployment

Web ini tidak akan selamanya disimpan sendiri. Saya berharap website ini dapat diakses oleh banyak orang dan semua dapat menikmati web aplikasi. Saya memutuskan untuk mempublikasikan web untuk menunjukkan rasa bangga dan saya yakin web ini bisa dikembangkan lebih baik.
